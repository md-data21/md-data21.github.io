Summary: Latitude vs. X

The purpose of this project was to use Python APIs and Pandas to show how weather patterns change as you get closer or further from the equator. First we choose a random set of 500+ cities around the world. Then we graphed multiple scatter plots comparing the latitude of the cities to four differnt weather factors:

  • Temperature (F) - as pictured above
  • Humidity %
  • Cloudiness %
  • Wind Speed (MPH)

You can find the raw data used by clicking the Data link in the navigation bar. Similarly, you can compare all the graphs to one another on the Comparison page. Finally, you can look the individaul scatter plots more closely by either choosing a scatter plot from the Plots drop down menu in the navigation bar OR by clicking on the individal graph under the Visualizations header on the right side of the page.

On these pages you will find more details around the plots and our findings.
